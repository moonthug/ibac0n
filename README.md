# iBac0n

A simple command line app for creating an iBeacon on a Mac.

Based heavily on [BeaconOSX](https://github.com/mttrb/BeaconOSX).

### Version
0.0.1


### Examples
```sh
# ibac0n <uuid> <major> <minor> <power>
./ibac0n 05b56fa9-b4b8-46f8-ae82-83587b5cc348 1 1 -60
```
